RPCs:

0x1: Null
0x2: Echo

0x10: Debug print

0x100: Announce
0x101: Query
0x102: Read
0x103: Write



Announce: num functions, type of each
Query: same return type

Read: function
Write: function, data



Sensors periodically send announce

SensorNet:
ListSensors()
GetSensor(id or name)



Sensor:
get()
set()
